# Image Generation Prompts — Landing Page Illustrations

## Global Style Guide (Apply to ALL images)

Before generating any image, read these rules carefully:

- **Style:** Flat illustration, clean vector-like aesthetic, minimal detail, no 3D, no photorealism, no gradients on people
- **Color Palette (STRICT — use ONLY these):**
  - Primary: `#1b4d5c` (dark teal-blue) — dominant color
  - Accent: `#dd6b4d` (warm coral/orange) — highlights and focal points
  - Background: `#f9f5ef` (warm beige/cream) — canvas
  - Light teal: `#e8f1f3` — subtle fills
  - Dark text: `#1f3640` — outlines if needed
  - White: `#ffffff` — paper/cards
- **People:** Middle-Eastern appearance, professional but approachable, modest clothing, NO suits/ties (think smart-casual: button-down shirt, no jacket). Men may have short beard. Women wear hijab naturally (not forced or exaggerated).
- **NO text in the image** — no Arabic, no English, no logos, no watermarks, no labels
- **NO AI artifacts:** no extra fingers, no melted objects, no floating disconnected elements
- **Aspect ratios** are specified per image
- **Transparent background** where specified, otherwise use `#f9f5ef`
- **Cultural sensitivity:** The context is Jordan. Avoid stereotypes. People should look like normal Jordanian professionals/entrepreneurs.

## Delivery Notes

- Export all images as **PNG** (except OG which is JPG)
- Maximum file size per image: **150KB** (compress if needed)
- If the AI model adds text to any image despite instructions, **regenerate** — no text is acceptable
- If the AI model generates photorealistic output, add "flat vector illustration style, 2D, no 3D rendering" to the prompt and retry
- After generating, check each image at **50% zoom** — if details are lost or muddy at that size, the illustration is too complex. Simplify and regenerate.
